## Abstract

Traditional asset pricing theory locates value in an asset's own attributes. Negation-based Economics proposes the opposite: value resides in relations of exclusion. An airspace route is valuable not because of what it IS, but because of what it NEGATES — a 20-minute flight replacing a 2-hour ground journey.

This paper establishes the axiomatic foundation of this proposition. Five core axioms form a generative chain from "parsing" to "value." We introduce the parsing operator, the negation channel, and channel closure as core mathematical objects. The axiomatic path is independent of historical transaction data, providing a unified mathematical foundation for treating emerging economic objects — low-altitude airspace, data assets, carbon emission allowances — not yet assetized under traditional frameworks.
